After making any changes to the file(s), follow the below steps to see the changes reflected on the hosted website. 

Steps:
1 - Open any terminal and write the below statements.
2 - to check the status of the files that have to be updated
    command: git status
    The file names will be highlighted in red if it hasn't been added to the commit.
3 - Add the files to the commit
    command: git add .
4 - To check if the files have been added to the commit.
    command: git status
    Once the files have been added to the commit its name will be highlighted in green.
5 - To commit the changes
    command: git commit -m "Title of the update" -m "description of the update"
6 - To push the changes to github
    command: git push
